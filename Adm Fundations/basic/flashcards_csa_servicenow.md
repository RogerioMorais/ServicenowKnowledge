# Flashcards - CSA ServiceNow

## Como usar
- Leia a pergunta e tente responder antes de ver a resposta.
- Revise os cartões em blocos curtos todos os dias.
- Foque primeiro nos conceitos mais cobrados na prova.

---

## Flashcards 1

### 1. O que é a ServiceNow?
Resposta: É uma plataforma de automação e transformação digital baseada em nuvem.

### 2. O que é uma Table?
Resposta: É a estrutura de armazenamento de dados.

### 3. O que é um Record?
Resposta: É uma linha da tabela, ou seja, um registro.

### 4. O que é um Field?
Resposta: É uma coluna da tabela.

### 5. O que é o sys_id?
Resposta: É o identificador único de cada registro.

---

## Flashcards 2

### 6. O que é um User?
Resposta: É uma pessoa que acessa a plataforma.

### 7. O que é um Group?
Resposta: É um conjunto de usuários usado para organização e atribuição.

### 8. O que é uma Role?
Resposta: É o conjunto de permissões que define o que um usuário pode fazer.

### 9. O que é uma ACL?
Resposta: É uma regra que controla o acesso a dados, registros e campos.

### 10. Qual a diferença entre Role e ACL?
Resposta: Role define permissões; ACL define se o acesso é permitido.

---

## Flashcards 3

### 11. Quais são os ambientes principais?
Resposta: Development, Test e Production.

### 12. O que é Development?
Resposta: Ambiente para criar e configurar soluções.

### 13. O que é Production?
Resposta: Ambiente real com dados operacionais.

### 14. O que é a PDI?
Resposta: É uma Personal Developer Instance usada para estudo e testes.

---

## Flashcards 4

### 15. O que é Configuration?
Resposta: É adaptar a plataforma com recursos nativos.

### 16. O que é Customization?
Resposta: É criar algo novo ou específico que não existe nativamente.

### 17. O que é um Reference Field?
Resposta: É um campo que aponta para outro registro em outra tabela.

### 18. O que é uma Transform Map?
Resposta: É o mecanismo que mapeia dados do Import Set para a tabela alvo.

---

## Flashcards 5

### 19. O que é uma List?
Resposta: É uma visualização com vários registros.

### 20. O que é um Form?
Resposta: É a tela de detalhamento de um registro.

### 21. O que é uma View?
Resposta: É uma forma de apresentar dados sem alterar os registros.

### 22. O que é um Filter?
Resposta: É uma busca que aplica critérios a uma lista de registros.

### 23. O que é Dot-Walking?
Resposta: É a navegação entre tabelas relacionadas.

## Flashcards 7

### 28. O que é um Catalog Item?
Resposta: É um item do catálogo que pode ser solicitado.

### 29. O que é uma Variable?
Resposta: É um campo dinâmico em um item de catálogo.

### 30. O que representa REQ → RITM → SCTASK?
Resposta: Pedido completo, item solicitado e tarefa de atendimento.

### 31. O que é One-Stop Shopping?
Resposta: É o conceito de centralizar a solicitação de serviços em um único ponto.

---

## Flashcards 8

### 32. O que são Notifications?
Resposta: São mensagens automáticas enviadas em resposta a eventos.

### 33. O que é o Outbox?
Resposta: É o local onde os e-mails enviados podem ser visualizados.

### 34. Notification substitui ACL?
Resposta: Não.

### 35. Usuário inativo recebe notification?
Resposta: Não, normalmente não.

---

## Flashcards 9

### 36. O que é um UI Policy?
Resposta: É uma regra que altera o comportamento visual e a apresentação do formulário.

### 37. O que é um Data Policy?
Resposta: É uma regra que garante a integridade dos dados.

### 38. O que é um Client Script?
Resposta: É um script executado no navegador.

### 39. O que é um Business Rule?
Resposta: É uma regra de lógica de negócio executada no servidor.

### 40. Qual é a diferença clássica entre UI Policy e Data Policy?
Resposta: UI Policy atua na interface; Data Policy garante o dado.

---

## Flashcards 10

### 41. O que é um Update Set?
Resposta: É um agrupamento de alterações para transporte entre instâncias.

### 42. O que é o ATF?
Resposta: É o Automated Test Framework para testes automatizados.

### 43. O que é o Shared Responsibility Model?
Resposta: É o modelo que divide responsabilidades de segurança entre ServiceNow e o cliente.

### 44. O que é Encryption At Rest?
Resposta: É a criptografia de dados armazenados.

### 45. O que é Encryption In Transit?
Resposta: É a criptografia de dados durante a transmissão.

### 46. Qual tecnologia protege dados em trânsito?
Resposta: TLS.
