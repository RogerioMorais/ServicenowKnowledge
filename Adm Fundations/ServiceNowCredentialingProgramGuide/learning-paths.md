# ServiceNow University Learning Paths

This document outlines the various learning paths offered by ServiceNow University, designed to help individuals develop in-demand skills and achieve professional goals.

## Key Information

*   **Pricing:** Effective February 4, 2025.
*   **Icon Legend:**
    *   `O`: No charge, On-Demand Course
    *   `2D`: 1,900 LC, 2-day ILT Course
    *   `3D`: 2,700 LC, 3-day ILT Course
*   **Abbreviations:**
    *   `ILT`: Instructor-led Training
    *   `CIS`: Certified Implementation Specialist
    *   `CSA`: Certified System Administrator
    *   `CAD`: Certified Application Developer
    *   `LC`: Learning Credit
*   **Navigation:** Select links on the right to navigate to specific learning paths. Use the "Return to ToC" icon to return to the table of contents.
